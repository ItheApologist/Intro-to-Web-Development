Labels in Forms

Labels allow us to add captions to our fields in our forms. Not only does this inform the regular user as to what each form field is requesting, this is also very important for visually impaired users. Assistance software looks for these labels when reading back to the user.

Below we see the same form we have been making. We have added label tags in order to add those captions to our form. All inputs are placed within the label tag in order to add the caption. See the source page for the code example.

Another method of using labels is to use a "for" attribute that corresponds to an "id" attribute used in the input tag.
