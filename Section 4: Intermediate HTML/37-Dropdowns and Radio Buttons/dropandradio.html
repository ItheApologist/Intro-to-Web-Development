New User Input Types

This example will cover three additional types of user inputs: radio buttons, select tags, textarea tags.

Below is an input type of "radio". Next to it is an input of type "checkbox" so you can view the difference. The major difference is that a radio tag cannot be turned off. As well, the radio button only allows for one choice at a time. In contrast, checkboxes allow the user to check off as many or little (including none) choices as they would like.

The below example contains a simpole questionair asking if you prefer cats or dogs. Obviously any sane person would chose cats, but we won't get into that now. What's important is that we have set up a radio button for each answer. Despite that, the user can select both making it all but useless at the moment.
