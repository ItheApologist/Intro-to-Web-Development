This is a first paragraph. Notice how the paragraph will end soon due to the use of the "p" tags.

Now the second paragraph has started. By starting with the "p" tage, this second set of text can take place seperate from the paragraph above. It's important to use tages like this to organize your documents in a clean fashion.

Remember, when writting HTML5 code, your structer and style need to be seperate. By this, structer tages use the traditional simple abbreviations like "p" for creating paragraphs. However, when you want to put text in BOLD, or, maybe want to set text in italics, don't use the tradition "b" for bold or "i" for italics. Use the new versions like "strong" for bold or "em" for italics (em is short for emphasize). These more discriptive versions help keep the code CLEAN by seperating the structer tags from the style tags, an important skill to have when people need to read your code!
